## Building and running on localhost

First install dependencies:

```sh
yarn
```

To run in hot module reloading mode:

```sh
yarn start
```

To run a production build:

```sh
yarn serve
```

To lint

```sh
yarn lint
```

To format

```sh
yarn prettier
```
